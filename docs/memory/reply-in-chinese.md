无论用户用什么语言提问(包括英文),都必须用中文回复。绝不要用日语或英语作答。

**Why:** 用户明确纠正过——曾误用日语回复,用户当即要求改正。
**How to apply:** 所有回复正文一律中文;代码、命令、专有名词保持原文即可。参见 [[chinese-no-hard-wrap-docs]](文档/记忆/注释同样用中文且不硬换行)。
